## 1. 特定商取引法（特商法）

### 対象となる取引

インターネットを通じたデジタルコンテンツの販売は、特定商取引法（通販規制）の対象となります。

### 必要な表記事項（主要項目）

| 項目 | 内容 |
|------|------|
| 販売事業者名 | 法人名または個人名・屋号 |
| 代表者氏名 | 法人の場合必須 |
| 所在地 | 住所（請求により開示も可） |
| 電話番号 | 請求により開示も可 |
| メールアドレス | 必須 |
| 商品の名称 | 必須 |
| 販売価格 | 税込で表示 |
| 支払方法 | 必須 |
| 商品の引き渡し時期 | 必須 |
| 返品・キャンセルポリシー | 必須 |

### 本キットでの対応

`/legal/tokushoho` ページにテンプレートを用意しています。
**必ず実際の情報に書き換えてください。**

`src/pages/legal/tokushoho.astro` を編集して、以下を実際の情報に変更する:
- 事業者名
- 代表者名
- 所在地
- 電話番号
- メールアドレス
- 販売URL

## 2. 消費税

### 個人事業主・フリーランスの場合

- **課税事業者**: 消費税の申告・納税義務あり
- **免税事業者**: 前々年度の課税売上が1,000万円以下の場合、消費税の納税義務なし

### インボイス制度（2023年10月〜）

- 課税事業者はインボイス（適格請求書）の発行義務
- 免税事業者のままでいる場合、取引先（法人）が仕入税額控除を受けられない場合がある
- デジタルコンテンツのB2C販売では、個人購入者相手が多く影響が限定的な場合もある
- **詳細は税理士に確認してください**

### Stripe Taxについて

Stripe Taxを有効にすると、購入者の所在地に応じた消費税を自動計算・収集できます。
設定・会計処理については税理士に相談を推奨します。

## 3. 著作権

### コンテンツの著作権

デジタルコンテンツ（テンプレート・マニュアル等）を販売する場合、著作権の帰属と利用許諾範囲を明確にする必要があります。

利用規約（`/legal/terms`）に以下を明記することを推奨:
- 著作権の帰属（販売者に帰属する旨）
- 許可される利用範囲（商業利用の可否）
- 禁止事項（再配布・転売の禁止等）

### 第三者コンテンツの利用

商品に他者のコンテンツ（画像・フォント・コードなど）を含める場合は、ライセンスを確認してください。
- フォント: 商用利用ライセンスの確認
- 画像: ストック画像の商用ライセンス確認
- オープンソースコード: ライセンス（MIT/Apache等）の確認

## 4. プライバシー保護

### 個人情報保護法

個人情報（氏名・メールアドレス・IPアドレス等）を収集・利用する場合、個人情報保護法の対応が必要です。

本キットのデフォルト設計:
- クリック計測でIPアドレスを収集しない
- Cookieを使用しない
- お問い合わせメールで収集した情報は対応のみに使用

### プライバシーポリシー

`/legal/privacy` にテンプレートを用意しています。
以下の点を実態に合わせて修正してください:
- 収集する情報の種類
- 利用目的
- 第三者への提供
- お問い合わせ先

## 5. 返金・キャンセルポリシー

### デジタルコンテンツの特性

デジタルコンテンツは「ダウンロード済み・閲覧済み」になると原状回復が困難であるため、多くのプラットフォームで返金不可ポリシーが認められています。

### 各プラットフォームのポリシー

各販路には独自の返金ポリシーがあります。
必ず公式ヘルプを確認し、それに従って対応してください。

- BOOTH: ショップ独自のポリシーを設定可能
- note: note のポリシーに従う
- Gumroad: Gumroad のポリシーに従う（購入者保護あり）
- BASE: BASE のポリシーに従う
- Stripe: 自分でポリシーを設定可能

### 本キットでの記載例

```
デジタルコンテンツの性質上、購入後の返金・キャンセルは原則対応しておりません。
ただし、ファイルが破損している、内容が説明と著しく異なるなどの重大な問題がある場合は
個別にご相談ください。
```

## 6. 海外販売（越境EC）の留意点

### 輸出規制

デジタルコンテンツの海外販売では、対象国によっては輸出規制（経済制裁国等）への注意が必要です。
Gumroad等は自動的に制裁対象国への販売をブロックする設定があります。

### 海外の消費税

EUではVATの対応が必要な場合があります（EU VATルール）。
Gumroad・Stripeは税の自動計算機能を持っています。

### 英語の利用規約

英語圏向けに販売する場合、利用規約も英語で用意することを推奨します。
