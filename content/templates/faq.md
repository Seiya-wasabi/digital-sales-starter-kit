## Q1. このキットは何が含まれますか？

**A.** 以下が含まれます：

- Astro製LP（静的サイト）ソースコード
- Cloudflare Pages Functions（クリック計測API）
- 出品原稿テンプレ（BOOTH / note / Gumroad / BASE / Stripe 各対応）
- SNS投稿テンプレ（X / note 各30本）
- SEOキーワード計画ドキュメント
- 月1時間運用ドキュメント
- 法的ページテンプレ（特商法・利用規約・プライバシー・免責）

## Q2. プログラミングの知識は必要ですか？

**A.** 基本設定はYAMLファイルの編集のみです。デプロイはCloudflare PagesのGitHub連携GUIから行えます。

## Q3. 月額費用はいくらかかりますか？

**A.** Cloudflare Pages無料プラン＋Workers KV無料枠だけで月$0で運用できます。
規模が大きくなったら Workers Paid ($5/月) への移行を検討してください。
